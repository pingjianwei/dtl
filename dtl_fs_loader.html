load_template_source/1

load_template_source(Name::list()) -> {ok, binary(), list()} | {error, no_template_dirs | not_found}

Searches configured template directories for the named template. Returns the file's contents if found, and its path, or an error if there was one.

load_template_source/2

load_template_source(Name::list(), Dirs::[list()]) -> {ok, binary(), list()} | {error, no_template_dirs | not_found}

Searches the specified directories for the named template (rather than the application's configured directory list). If the provided list is empty, we use the configured list of these.

If the template is not found, this returns a not_found error.
