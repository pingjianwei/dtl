Libraries are containers of template tags and filters.

This module defines the dtl_library behaviour.
Required callback functions: registered_filters/0, registered_tags/0.

Description

Libraries are containers of template tags and filters. Custom DTL tags and filters must be implemented in a module that implements the dtl_library behaviour.
